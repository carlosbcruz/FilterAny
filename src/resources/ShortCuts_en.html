Shortcut List


General Application Shortcuts
Shortcut Description
F1 Show search filter dialog.
Alt + E Execute the current filter.
Alt + X Show the execution exemple of the current filter.
Alt + L Show the help of the current filter.
Alt + C Compare Main with Output Area.
Alt + S Switch the content between the Main and Output Area.
Alt + T Toggle between text and file mode.
Alt + A Toggle automatic filter execution on and off.
Alt + B Open scrap book.
Alt + M Maximize FilterAny.
ESC Minimize FilterAny.
Ctrl + Q Exit FilterAny.

Text Editor Specific Shorcuts
Shortcut Description
Ctrl + D Open the formatted editor.
Ctrl + X Cut the selected text.
Ctrl + C Copy the selected text.
Ctrl + V Paste the selected text.
Ctrl + F Open the Find Text dialog.
Ctrl + G Open the Go To Line dialog.
Ctrl + Z or Alt + Z Undo last text change.
Ctrl + Y or Alt + Y Redo last text change.
Alt + U Update the counter of the selected area.
Alt + D Toggle the line feed of the selected area between Unix and DOS format.
