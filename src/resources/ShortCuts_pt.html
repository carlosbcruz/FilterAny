Lista de Atalhos


Atalhos Gerais da Aplicação
Atalho Descrição
F1 Abre a janela de busca por filtros.
Alt + E Executa o filtro corrente.
Alt + X Mostra o exemplo de execução do filtro corrente.
Alt + U Mostra a ajuda do filtro corrente.
Alt + C Compara a Área Principal com a de Saída.
Alt + T Troca o conteúdo entre a Área Principal e a Auxiliar.
Alt + I Troca entre o modo de texto e o de arquivo.
Alt + L Troca entre modo de execução manual ou automático.
Alt + H Visualizar o rascunho.
Alt + M Maximiza o FilterAny.
ESC Minimiza o FilterAny.
Ctrl + Q Sair do FilterAny.

Atalhos Específicos do Editor de Texto
Atalho Descrição
Ctrl + D Abre o editor com formatação.
Ctrl + X Recorda o texto selecionado.
Ctrl + C Copia o texto selecionado.
Ctrl + V Cola o texto que está na área de transferência.
Ctrl + L Abre o diálogo para procurar conteúdos.
Ctrl + G Abre o diálogo para ir a uma linha específica.
Ctrl + Z or Alt + Z Desfaz a última alteração no texto.
Ctrl + Y or Alt + Y Resfaz a última alteração no texto desfeita.
Alt + A Atualiza os contador da área de texto selecionada.
Alt + D Trocar entre o fim de linha do formato Unix e DOS.

Atalhos de Menu
Atalho Descrição
Alt + P, Enter Compara a Área Principal com a de Saída.
Alt + P, C, Enter Compara a Área Principal com a Auxiliar.
Alt + P, C, C, Enter Compara a Área Auxiliar com a de Saída.
Alt + R, EnterSalvar Área Principal.
Alt + R, S, EnterSalvar Área Auxiliar.
Alt + R, S, S, EnterSalvar Área de Saída.
Alt + R, A Sair do FilterAny.
